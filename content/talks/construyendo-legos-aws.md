## Descripción de la charla

Construyendo Legos con AWS fue una plática que impartí con el AWS Cloud Club UTP el 17 de Septiembre de 2025.

Esta sesión invita a reflexionar sobre la imaginación de tener un enorme set de Legos para diseñar cualquier cosa que imagines: ¡eso es AWS!

### AWS como bloques de construcción

AWS nos ofrece bloques tecnológicos listos para ensamblar y crear soluciones en la nube.

En esta sesión hablo de cómo podemos usar sus piezas –los servicios– para construir desde pequeñas aplicaciones hasta proyectos que impacten a millones de personas.

### Una invitación a crear

Una invitación a experimentar, aprender y crear en la nube, usando los servicios de AWS como piezas de Lego que se conectan entre sí para dar vida a tus ideas.
